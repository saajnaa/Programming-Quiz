# Coding Test Website

A simple React-based quiz application to test your knowledge of HTML, CSS, and JavaScript. The app features multiple-choice questions for each topic and a clean, responsive UI.

## Features

- Quiz sections for HTML, CSS, and JavaScript
- Navigation bar for easy topic selection
- Responsive design for desktop and mobile
- Built with React and React Router

## Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   cd coding-test-website
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```

### Usage

- Navigate through the quiz using the navigation bar.
- Select an answer for each question and submit your response.
- Review your score at the end of each quiz section.
